  <action>
RED: Create tests/test_delay_line.py with all tests above. Import `from claudeverb.algorithms.filters import DelayLine`. Tests must fail initially.

GREEN: Implement DelayLine in claudeverb/algorithms/filters.py following the research pattern:
- `__init__(self, max_delay: int)`: Allocate `np.zeros(max_delay, dtype=np.float32)` buffer, `_write_index = 0`, store `_max_delay`.
- `write(self, sample: float) -> None`: Store sample at write_index as np.float32, advance write_index with modulo wrap.
- `read(self, delay: float) -> float`: Calculate read position as `_write_index - delay`, compute integer index and fractional part, interpolate between two adjacent samples using `buffer[idx0] + frac * (buffer[idx1] - buffer[idx0])`. Both indices wrapped with modulo.
- `process(self, block: np.ndarray) -> np.ndarray`: Check `block.dtype != np.float32`, raise TypeError if not. Loop over samples calling read then write. Return float32 output array.
- `reset(self) -> None`: Zero buffer, reset write_index to 0.

Convention: In each sample cycle of process(), read BEFORE write (so delay of N means output appears N samples after input).

Handle edge case: When `delay` equals `max_delay`, it reads the oldest sample in the buffer.
