    Key implementation details from research:
    - Input diffuser AP lengths at 29761: [142, 107, 379, 277], coeffs [0.75, 0.75, 0.625, 0.625]
    - Decay AP1 lengths: [672, 908], coeff: -decay_diffusion1 (NEGATIVE)
    - Tank delay1 lengths: [4453, 4217] (modulated by LFO)
    - Decay AP2 lengths: [1800, 2656], coeff: +decay_diffusion2
    - Tank delay2 lengths: [3720, 3163]
    - LFO: sine at 1 Hz, phase_inc = 2*pi*1.0/48000. Right half uses lfo_phase + pi for 180-deg offset
    - Output taps from paper Table 2 (see RESEARCH.md Pattern 3 for exact positions)
    - OnePole: y = x * coeff + y_prev * (1 - coeff)
    - DCBlocker: y = x - x_prev + 0.99935 * y_prev
    - Use math.sin() not numpy (C portability)
    - All tap positions must be scaled from 29761 to 48 kHz
