  <files>claudeverb/audio/impulse.py, claudeverb/audio/__init__.py, tests/test_impulse_response.py</files>
  <behavior>
    - generate_impulse_response(algorithm) returns float32 mono array with decaying reverb tail
    - generate_impulse_response(algorithm, mono=False) returns float32 stereo (2, N) array
    - Output length = duration_seconds * SAMPLE_RATE (default 3.0s = 144000 samples)
    - Algorithm is reset before processing (deterministic results)
    - Two calls with same algorithm and params produce identical output
    - Output has non-zero energy beyond sample 0 (reverb tail captured)
    - Output energy decays toward zero near the end (tail captured fully for typical settings)
  </behavior>
  <action>
    Create `claudeverb/audio/impulse.py`:
    - `generate_impulse_response(algorithm: ReverbAlgorithm, duration_seconds: float = 3.0, mono: bool = True) -> np.ndarray`:
      1. Compute n_samples = int(duration_seconds * SAMPLE_RATE)
      2. Create impulse: zeros of length n_samples, set [0] = 1.0, dtype float32
      3. If not mono: stack to (2, N) stereo impulse
      4. algorithm.reset()
      5. result = algorithm.process(impulse)
      6. Return result

    Update `claudeverb/audio/__init__.py` to export generate_impulse_response.

    Create `tests/test_impulse_response.py`:
    - test_ir_mono_shape_dtype: Generate mono IR from Freeverb, verify shape (N,), dtype float32, length correct
    - test_ir_stereo_shape: Generate stereo IR, verify shape (2, N)
    - test_ir_has_reverb_tail: Verify samples beyond index 1000 have nonzero energy (reverb tail)
    - test_ir_deterministic: Generate twice with same Freeverb instance, output identical (reset works)
    - test_ir_custom_duration: Generate 1.0s IR, verify length = 48000
    - test_ir_tail_decays: Verify last 10% of IR has lower RMS than first 10% (tail is decaying)
  </action>
