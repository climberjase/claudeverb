  <name>Task 2: Implement FDNCore class with Hadamard mixing, damping, modulation, and DC blocking</name>
  <files>claudeverb/algorithms/fdn_reverb.py</files>
  <action>
    Create claudeverb/algorithms/fdn_reverb.py containing:

    **Module-level constants:**
    - BASE_DELAYS = [601, 773, 1009, 1249] (all prime, coprime, ~12-26ms at 48kHz)
    - SAMPLE_RATE = 48000
    - MAX_MOD_EXCURSION = 16 (samples, safety margin for LFO)

    **hadamard_4(a, b, c, d) function:**
    - Fast Walsh-Hadamard Transform butterfly implementation
    - Two stages: pairs then cross-pairs, normalize by 0.5
    - Returns tuple of 4 floats

    **rt60_to_gain(rt60_seconds, delay_samples, sample_rate=48000) function:**
    - Converts RT60 decay time to per-loop feedback gain
    - Formula: 10^(-3 * delay_samples / (rt60_seconds * sample_rate))
    - Returns 0.0 if rt60_seconds <= 0
    - Clamp result to max 0.9999 to prevent instability

    **FDNCore class:**
    - __init__(self, base_delays: list[int], sample_rate: int = 48000):
      - Create 4 DelayLine instances with max_delay = base_delay + MAX_MOD_EXCURSION + 2
      - Create 4 OnePole damping filters (default coefficient 0.3)
      - Create 4 DCBlocker instances
      - Initialize LFO state: _lfo_phase=0.0, _lfo_phase_inc for 0.7 Hz
      - Store _base_delays, _current_delays (scaled by size), _channel_gains
      - Call reset()

    - process_sample(self, inputs: list[float]) -> list[float]:
      - Read from each delay line with LFO modulation (alternating phase per channel: channels 0,2 get +lfo, channels 1,3 get -lfo)
      - Clamp modulated read delay to max(1.0, delay)
      - Advance LFO phase (wrap at 2*pi)
      - Per-channel: damping filter -> DC blocker -> multiply by channel gain
      - Hadamard butterfly mixing on the 4 processed outputs
      - Write back to delay lines: input[ch] + mixed[ch]
      - Return the PRE-mixing outputs (these are the tap points for stereo output)

    - set_decay(self, rt60_seconds: float) -> None:
      - Store _rt60
      - Recompute _channel_gains using rt60_to_gain() for each channel's current delay length

    - set_damping(self, coefficient: float) -> None:
      - Set all 4 OnePole filter coefficients (0.0 = no damping/bright, 1.0 = full damping/dark)

    - set_modulation(self, depth: float, rate: float) -> None:
      - Set _mod_depth (in samples, 0 to MAX_MOD_EXCURSION)
      - Set _lfo_phase_inc = 2*pi*rate/sample_rate

    - set_size(self, scale: float) -> None:
      - Scale _current_delays = [int(d * scale) for d in _base_delays], clamped to min 1
      - Recompute channel gains to maintain current RT60 at new delay lengths

    - set_freeze(self, frozen: bool) -> None:
      - If frozen: set all channel gains to 1.0 (sustain forever)
      - If not frozen: recompute gains from current RT60

    - reset(self) -> None:
      - Reset all delay lines, damping filters, DC blockers
      - Reset LFO phase to 0.0
      - Reset _current_delays to _base_delays

    **C struct comment at top of file:**
    Start with a C struct typedef comment showing the equivalent C representation, following the pattern in freeverb.py and dattorro_plate.py. Include: 4 delay line buffers, 4 damping filter states, 4 DC blocker states, LFO state, gains, delay lengths.
