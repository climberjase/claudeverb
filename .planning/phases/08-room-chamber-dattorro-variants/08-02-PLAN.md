  <action>
Create `claudeverb/algorithms/dattorro_single_loop.py` -- fully independent implementation (NOT subclassed from DattorroPlate per CONTEXT.md decision):

1. Start with C struct typedef comment at top.

2. Define constants for the single-loop topology at 48 kHz (from research Pattern 3):
   - 4 allpass sections in loop: base delays ~[240, 384, 528, 336] samples (~5ms, 8ms, 11ms, 7ms)
   - 4 delay sections between allpasses: base delays ~[720, 1056, 864, 1200] samples (~15ms, 22ms, 18ms, 25ms)
   - Allpass feedback coefficients: 0.6
   - Input diffusion: 4 allpass diffusers (like DattorroPlate but own lengths, e.g., [142, 107, 379, 277] scaled from 29761Hz to 48kHz)

3. Implement `DattorroSingleLoop(ReverbAlgorithm)`:
   - `__init__`: Create input diffuser chain (4 AllpassFilters), 4 loop AllpassFilters, 4 loop DelayLines (max_delay sized for Loop Length=100 scaling), 2 OnePole damping filters (at positions after D1 and D3 in loop), 1 DCBlocker at loop junction, LFO state for modulation on D1, pre-delay DelayLine
   - Store base delay values for Loop Length scaling

4. `param_specs`: Per CONTEXT.md knob mapping:
   - decay (0-100), loop_length (0-100, default 50), damping (0-100), mod (0-100, default 30), mix (0-100, default 50), pre_delay (0-100)
   - switch_1: Freeze(-1) / Normal(0) / Shimmer(1)
   - switch_2: Mono(-1) / Stereo(0) / Wide(1)

5. `_scale_params`: Loop Length 0-100 maps to scale 0.375-1.875 (30ms to 150ms total loop at scale=1.0 ~80ms base). Decay maps to 0.5-10.0s RT60. Damping maps OnePole coeff 1.0-0.05 (inverted). Mod maps depth 0-1.0 samples and rate 0.1-2.0 Hz.

6. `update_params`: Apply Loop Length scale to all 8 delay lengths (4 AP + 4 delay). Compute feedback gain using rt60_to_gain() with total loop delay. Apply damping. Apply modulation settings. Handle freeze (bypass damping/DC/modulation, boost gain).

7. `_process_impl(audio)`: Sample-by-sample:
   - Mono input -> pre-delay -> input diffusers (4 allpass cascade)
   - Single-loop processing per research Pattern 3:
     - Read feedback from end of loop (D4), apply DC blocker, multiply by feedback gain
     - Inject input + feedback
     - Section 1: AP1 -> write/read D1 (with LFO modulation) -> damping1
     - Section 2: AP2 -> write/read D2
     - Section 3: AP3 -> write/read D3 -> damping2
     - Section 4: AP4 -> write/read D4
   - Stereo output taps from multiple points in loop (research gives specific tap positions)
   - Width processing (Mono/Stereo/Wide switch)
   - Mix dry/wet, clip to [-1, 1]
   - Shimmer: pitch-shift feedback by octave (same approach as DattorroPlate if it has shimmer, otherwise use sample-skipping in delay read)

8. `reset()`: Reset all delay lines, allpass filters, OnePoles, DCBlocker, LFO state. Re-apply current params.

9. `to_c_struct()` and `to_c_process_fn()`: C code strings following existing pattern.

**CRITICAL stability notes (from research Pitfall 5):**
- Feedback gain must use rt60_to_gain() with TOTAL loop delay (sum of all 8 delay lengths)
- Clamp total feedback gain to max 0.999
- Apply damping at 2 points in loop, not just 1
- DC blocker at loop junction is essential

Create `tests/test_dattorro_variants.py` with TestSingleLoop class. Also create TestTripleDiffuser and TestAsymmetric stubs (skip markers) for Plan 04.
