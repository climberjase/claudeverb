   - Freeze: bypass damping/DC/modulation in tank, boost output gain
   - Shimmer: pitch-shift in feedback (same approach as Single-Loop or Plate)

Create `claudeverb/algorithms/dattorro_asymmetric.py` -- fully independent implementation:

1. C struct typedef comment at top.

2. Define base figure-eight tank values (same as Plate, scaled to 48kHz). The Spread knob (0-100) morphs asymmetry:

   ```
   spread_factor = spread / 100.0  # 0.0 to 1.0
   # Left half: shorter delays
   left_scale = 1.0 - spread_factor * 0.3    # 1.0 to 0.7
   # Right half: longer delays
   right_scale = 1.0 + spread_factor * 0.3   # 1.0 to 1.3
   # Max ratio: 1.3/0.7 = ~1.86 (under 2.0 limit from Pitfall 6)
   # But clamp to 1.6:1 as research recommends: actually 1.3/0.7 ~ 1.86, too high
   # Better: left_scale = 1.0 - spread_factor * 0.2, right_scale = 1.0 + spread_factor * 0.25
   # Ratio at max: 1.25/0.8 = 1.5625 (safely under 1.6)
   ```

3. Implement `DattorroAsymmetric(ReverbAlgorithm)`:
   - `__init__`: Create 4 input AllpassFilters (standard Dattorro input diffusion), asymmetric figure-eight tank where L and R half-loops have independently scaled delays
   - Diffuser count asymmetry per research Pattern 5 and Open Question 3:
     - At Spread < 50: both halves use 2 decay diffusers (like standard Dattorro)
     - At Spread >= 50: left uses 1 decay diffuser, right uses 2 (the "removed" left diffuser has its coefficient crossfaded to 0 for smooth transition)

4. `param_specs`: Per CONTEXT.md:
   - decay (0-100), spread (0-100, default 50), damping (0-100), mod (0-100, default 30), mix (0-100, default 50), pre_delay (0-100)
   - switch_1: Freeze(-1) / Normal(0) / Shimmer(1)
   - switch_2: Mono(-1) / Stereo(0) / Wide(1)

5. `update_params`: When Spread changes, recalculate L and R delay lengths from base values * left_scale/right_scale. Handle diffuser count transition (crossfade coefficient). Compute feedback gains separately for L and R based on their total path delays using rt60_to_gain().

6. `_process_impl`: Sample-by-sample:
   - Mono -> pre-delay -> input diffusion (4 allpass) -> asymmetric figure-eight tank -> stereo output taps -> width -> mix -> clip
   - L half-loop: decay_diffuser_L1 -> delay_L1 -> damping_L -> [optional decay_diffuser_L2 based on spread] -> delay_L2 -> cross-feed to R
   - R half-loop: decay_diffuser_R1 -> delay_R1 -> damping_R -> decay_diffuser_R2 -> delay_R2 -> cross-feed to L
   - Feedback gains: separately computed for L and R total path delays

7. `reset()`, `to_c_struct()`, `to_c_process_fn()` per standard pattern.

8. Create presets in `dattorro_asymmetric_presets.py`:
   - "Wide Plate" -- moderate spread (60), balanced decay (like standard plate but wider)
   - "Ultra-Wide" -- max spread (100), long decay, wide switch on (maximum stereo effect)
   - "Subtle Width" -- low spread (25), short decay (close to standard plate, slight widening)
