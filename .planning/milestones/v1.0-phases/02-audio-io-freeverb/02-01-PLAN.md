  <files>claudeverb/audio/__init__.py, claudeverb/audio/io.py, tests/test_audio_io.py</files>
  <behavior>
    - load() returns float32 ndarray at 48 kHz for any supported format (WAV, AIFF, FLAC, OGG)
    - load() returns shape (N,) for mono, (2, N) for stereo
    - load() handles >2 channel files by taking first two channels
    - load() always returns float32 (never float64, even from librosa edge cases)
    - save() writes WAV file readable by soundfile, default 32-bit float subtype
    - save() transposes (2, N) stereo to (N, 2) for soundfile convention
    - save() with subtype="PCM_24" produces 24-bit PCM output
    - Round-trip: load(save(audio)) preserves data within float32 precision
  </behavior>

    **Analysis signals (for testing and Phase 3 metrics):**
    - `generate_impulse_click(duration_s=1.0) -> np.ndarray`: Unit impulse at sample 0, zeros after. Length = int(SAMPLE_RATE * duration_s).
    - `generate_white_noise_burst(duration_s=0.5, burst_s=0.1) -> np.ndarray`: White noise burst (amplitude 0.5) for first burst_s seconds, silence after. Use numpy RNG with seed 42 for reproducibility.
    - `generate_sine_sweep(duration_s=2.0, f_start=20.0, f_end=20000.0) -> np.ndarray`: Logarithmic chirp via scipy.signal.chirp, amplitude 0.8. Cast to float32.

    **Instrument-like clips (for listening evaluation per AIO-02 "dry drums, guitar, vocals"):**
    - `generate_drum_transient(duration_s=0.5) -> np.ndarray`: Synthesized kick/snare-like transient. Sharp attack using a short burst of band-limited noise (200-8000 Hz via scipy.signal.butter bandpass) with fast exponential decay envelope (tau ~0.02s). Amplitude 0.9. This produces a punchy, percussive hit useful for evaluating reverb onset and early reflections.
    - `generate_guitar_pluck(duration_s=1.5, frequency=220.0) -> np.ndarray`: Karplus-Strong synthesis -- initialize a delay buffer (length = SAMPLE_RATE / frequency) with random noise, then iteratively average adjacent samples with a decay factor of 0.996. Run for duration_s * SAMPLE_RATE samples. This produces a realistic plucked-string timbre. Normalize peak to 0.8. Cast to float32.
    - `generate_vocal_tone(duration_s=2.0, fundamental=180.0) -> np.ndarray`: Synthesized vowel-like tone. Generate a sawtooth wave at fundamental frequency, then apply formant filtering: two bandpass biquad filters centered at ~700 Hz (F1, bandwidth 130 Hz) and ~1200 Hz (F2, bandwidth 70 Hz) to approximate an "ah" vowel. Use scipy.signal.sosfilt with second-order sections. Apply gentle amplitude envelope (0.1s attack, 0.2s release). Normalize peak to 0.7. Cast to float32.
