    Class Freeverb(ReverbAlgorithm) with __slots__.

    **Constants (class-level):**
    - COMB_LENGTHS_44100 = [1557, 1617, 1491, 1422, 1356, 1277, 1188, 1116]
    - ALLPASS_LENGTHS_44100 = [225, 556, 441, 341]
    - STEREO_SPREAD_44100 = 23
    - FIXED_GAIN = 0.015
    - SCALE_ROOM = 0.28, OFFSET_ROOM = 0.7
    - SCALE_DAMP = 0.4

    **_scale_length(length_44100):** round(length_44100 * SAMPLE_RATE / 44100)

    **__init__():** Set default param values (room_size=75, damping=25, mix=75, width=100, pre_delay=0, hf_damp=0, switch1=0, switch2=0). Call _initialize().

    **_initialize():** Create 8 CombFilters for L channel using scaled lengths. Create 8 CombFilters for R channel using scaled lengths + scaled STEREO_SPREAD. Create 4 AllpassFilters for L and 4 for R (same pattern). Create DelayLine for pre-delay (max_delay = ceil(0.1 * SAMPLE_RATE) = 4800). Call _scale_params().

    **6th knob -- hf_damp (High-Frequency Damping):** A secondary damping control. At 0, no extra HF damping. At 100, adds additional damping to the comb filter damp coefficients. Per user decision, the 6th knob is TBD -- using hf_damp as a useful independent HF control that differs from the main damping knob. Main damping controls overall feedback damping; hf_damp adds on top for extra HF roll-off. Implementation: `effective_damp = base_damp + (hf_damp_scaled * (1.0 - base_damp))` -- this blends from base_damp toward 1.0 (maximum damping).

    **_scale_params():** Convert 0-100 knob values to internal floats per RESEARCH.md formulas. Apply room_scaled as feedback to all combs. Apply effective damp (base damp + hf_damp contribution) to all combs. Calculate pre_delay_samples.

    **_process_impl(audio):** Dispatch to _process_mono or _process_stereo based on audio.ndim.

    **_process_mono(audio):** Per RESEARCH.md code example. Sample-by-sample loop: apply FIXED_GAIN, pre-delay (if samples > 0), sum 8 parallel combs, series 4 allpasses, mix dry/wet. Hard clip output to [-1.0, 1.0]. Handle freeze mode: when switch1==-1, set input gain to 0.0 (x = 0.0 after pre-delay).

    **_process_stereo(audio):** Per RESEARCH.md stereo code example. Mono sum input for reverb. Separate L/R comb+allpass chains. Width crossmix: wet1 = wet*(1+width)/2, wet2 = wet*(1-width)/2. Handle stereo_mode switch: -1 = force mono output (copy L to R after processing), 0 = normal, 1 = wide (double the width_scaled, capped at 1.0). Hard clip output.

    **update_params(params):** Update stored param values from dict, call _scale_params().
    **reset():** Call reset() on all 16 combs, 8 allpasses, pre-delay DelayLine.
    **param_specs property:** Return dict with all 6 knobs and 2 switches per RESEARCH.md Pattern 3.
