Biquad class with Direct Form II Transposed implementation:
- `__init__(self, b0, b1, b2, a1, a2, sample_rate=48000)`: Store all coefficients as np.float32. Initialize d1=0, d2=0 as np.float32. Store sample_rate.
- `process_sample(self, x: float) -> float`: DF2T equations: y = b0*x + d1; d1 = b1*x - a1*y + d2; d2 = b2*x - a2*y. Return float(y).
- `process(self, block: np.ndarray) -> np.ndarray`: Float32 enforcement, loop over process_sample.
- `reset(self) -> None`: Zero d1 and d2.
- `set_coefficients(self, b0, b1, b2, a1, a2) -> None`: Update coefficients as np.float32 WITHOUT resetting d1/d2.

Factory classmethods using Bristow-Johnson Audio EQ Cookbook formulas (all normalize by a0):
- `lowpass(cls, freq, q, sample_rate=48000)`: b0=(1-cos_w0)/2, b1=1-cos_w0, b2=(1-cos_w0)/2
- `highpass(cls, freq, q, sample_rate=48000)`: b0=(1+cos_w0)/2, b1=-(1+cos_w0), b2=(1+cos_w0)/2
- `notch(cls, freq, q, sample_rate=48000)`: b0=1, b1=-2*cos_w0, b2=1
- `bandpass(cls, freq, q, sample_rate=48000)`: b0=alpha, b1=0, b2=-alpha
- `parametric(cls, freq, q, gain_db, sample_rate=48000)`: A=10^(gain_db/40), b0=1+alpha*A, b1=-2*cos_w0, b2=1-alpha*A, a0=1+alpha/A, a1=-2*cos_w0, a2=1-alpha/A

Where w0 = 2*pi*freq/sample_rate, alpha = sin(w0)/(2*q), cos_w0 = cos(w0). Use `import math` for trig (not numpy -- maps to C math.h).

IMPORTANT: No scipy import in filters.py. All coefficient math is hand-coded.
